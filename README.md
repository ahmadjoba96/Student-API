# Student API

A lightweight REST API built with **Node.js** and **Express** as a practical task for the **AHL Logics** internship (Backend Session 1). 

The project focuses on backend fundamentals like routing, HTTP methods, and handling JSON data using temporary, in-memory storage.

## Features
- **GET /**: Returns a simple welcome message.
- **GET /students**: Retrieves the full list of students.
- **GET /students/:id**: Fetches a single student by ID.
- **POST /students**: Adds a new student (with name and age validation).
- **DELETE /students/:id**: Deletes a student by ID (Bonus challenge).

## Tech Stack
- Node.js
- Express.js
